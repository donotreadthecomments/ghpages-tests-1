Core Team

The Jekyll Core Team’s responsibility is to ensure the development and community around the Jekyll ecosystem thrive.

Plugin Core

The Jekyll Plugin Core Team’s responsibility is to ensure the development and community around the core plugins thrive. They also provide guidance in conversations about extensibility of Core Jekyll.
